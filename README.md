## 🔥 Features
* 🎯 ATS Match Score
* 🛠 Skill Gap Analysis
* 📊 Resume Evaluation (Skills, Experience, Keywords)
* 🚀 Improvement Suggestions
* 🎯 Best Role Recommendations
* 📄 Downloadable Report

---

## 🛠 Tech Stack

* Python
* Streamlit
* Groq (LLM API)
* PDFPlumber

---

## 🧠 How It Works

1. Upload your resume
2. Paste job description
3. AI analyzes match score
4. Get strengths, gaps, and improvements

---

## 💡 Why This Project?

Recruiters use structured evaluation systems.
This tool simulates that process using AI.
